# 🖥️ 일정 관리를 위한 Daybin
이 앱은 여러분을 위한 편리한 일정 관리 도구입니다.

Todo 리스트와 캘린더를 결합하여 여러분의 스케줄을 한눈에 파악할 수 있습니다.
<br>

## 🕰️ 개발 기간
* 24.04.15 - 22.05.08

### ⚙️ 개발 환경
- `Swift 5`
- `Xcode 15.3`
- **Framework** : SwiftUI , CoreData
